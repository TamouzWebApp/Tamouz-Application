# إعداد قواعد الأمان في Firebase

## 🔥 خطوات إعداد Firebase Realtime Database

### 1. الدخول إلى Firebase Console
1. اذهب إلى [Firebase Console](https://console.firebase.google.com/)
2. اختر مشروعك `tamouz-webapp`

### 2. إعداد Realtime Database
1. من القائمة الجانبية، اختر **Realtime Database**
2. إذا لم تكن قد أنشأت قاعدة بيانات، اضغط **Create Database**
3. اختر المنطقة المناسبة (مثل `us-central1`)
4. اختر **Start in test mode** مؤقتاً

### 3. تحديث قواعد الأمان
1. اذهب إلى تبويب **Rules**
2. استبدل القواعد الموجودة بالقواعد التالية:

```json
{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null",
    "events": {
      ".read": "auth != null",
      ".write": "auth != null",
      "$eventId": {
        ".validate": "newData.hasChildren(['id', 'title', 'date', 'time', 'location', 'description', 'category', 'priority', 'status', 'createdAt'])"
      }
    },
    "settings": {
      ".read": "auth != null",
      ".write": "auth != null"
    }
  }
}
```

3. اضغط **Publish** لحفظ القواعد

### 4. تفعيل المصادقة المجهولة
1. من القائمة الجانبية، اختر **Authentication**
2. اذهب إلى تبويب **Sign-in method**
3. اختر **Anonymous**
4. فعّل **Enable**
5. اضغط **Save**

## ✅ التحقق من الإعداد

بعد تطبيق هذه الخطوات:

1. **أعد تحميل الصفحة**
2. تحقق من مؤشر الحالة في الزاوية اليسرى - يجب أن يظهر "متصل ومصادق"
3. جرب إضافة حدث جديد
4. تحقق من أن البيانات تظهر في Firebase Console

## 🔧 استكشاف الأخطاء

### إذا ظهر "غير مصادق":
- تأكد من تفعيل المصادقة المجهولة
- أعد تحميل الصفحة
- تحقق من وحدة تحكم المطور للأخطاء

### إذا ظهر "PERMISSION_DENIED":
- تأكد من تحديث قواعد الأمان
- تأكد من أن القواعد تسمح للمستخدمين المصادقين بالقراءة والكتابة

### إذا لم تظهر البيانات:
- تحقق من أن قاعدة البيانات في نفس المنطقة
- تأكد من صحة رابط قاعدة البيانات في الإعدادات

## 📱 ميزات النظام الجديد

- ✅ **مصادقة تلقائية**: تسجيل دخول مجهول تلقائي
- ✅ **تحديثات فورية**: مزامنة البيانات في الوقت الفعلي
- ✅ **أمان محسن**: قواعد أمان محكمة
- ✅ **مؤشرات الحالة**: معرفة حالة الاتصال والمصادقة
- ✅ **معالجة الأخطاء**: رسائل خطأ واضحة ومفيدة
